#  **PLAN DE TRABAJO PARA TU PROYECTO FINAL (PASO A PASO)**  
Categoría: *Belleza y Cuidado Personal*

---

#  **FASE 1 — Estructura base del proyecto (HTML + carpetas)**

### 1. Crea la estructura del proyecto
Carpetas recomendadas:

```
/css
/js
/img
/php
```

### 2. Crea el archivo principal `index.html`
Incluye **solo la estructura general**:

- `<header>` → Menú superior  
- `<section id="carousel">` → Carrusel de productos  
- `<main id="principal">` → Información dinámica  
- `<footer>` → Información final  

 **No metas contenido todavía**, solo deja los contenedores listos.

---

#  **FASE 2 — Base de datos + servidor (PHP + MySQL)**

### 3. Crea la base de datos con dos tablas:

### **Tabla productos**
Campos mínimos:
- id (PK)
- nombre
- descripcion
- precio
- disponibilidad
- imagen (ruta)

### **Tabla usuarios**
Campos:
- id (PK)
- nombre
- email
- telefono
- cuenta_bancaria
- fecha_registro

### 4. Crea los endpoints PHP (AJAX)
En `/php/` crea:

- `getProductos.php` → devuelve JSON con productos  
- `getProducto.php?id=` → devuelve un producto  
- `registrarUsuario.php` → inserta usuario  
- `login.php` → valida usuario  
- `getCliente.php` (opcional)

 **No conectes aún con Vue**, solo asegúrate de que devuelven JSON correctamente.

---

#  **FASE 3 — Carrusel automático (JS puro o Vue)**

### 5. Implementa el carrusel
- Recupera productos con AJAX desde `getProductos.php`
- Muestra al menos 5 imágenes
- Haz que se deslicen automáticamente cada X segundos
- Al hacer clic → mostrar detalles en la división principal

---

#  **FASE 4 — VueJS3 (la parte más importante)**

Aquí empiezas a integrar Vue para manejar toda la lógica dinámica.

### 6. Crea la instancia principal de Vue
En `/js/app.js`:

- Datos: productos, productoSeleccionado, usuario, carrito, idioma, etc.
- Propiedades calculadas: total del carrito, productos filtrados…
- Métodos: agregarCarrito, eliminarCarrito, cambiarIdioma, login, logout…

### 7. Crea al menos **un componente con plantilla**
Ejemplo: `<producto-card>`  
- Recibe props: nombre, imagen, precio  
- Usa plantilla con `<template>`  
- Usa v-bind, v-on, v-if, v-for

### 8. Implementa todas las directivas obligatorias
- `v-if` → mostrar login o usuario  
- `v-for` → listar productos  
- `v-bind` → atributos dinámicos  
- `v-on` → eventos (click, submit)  
- `v-model` → formularios

### 9. CSS dinámico
Ejemplo:  
- Producto sin stock → clase roja  
- Producto seleccionado → borde especial  
- Idioma activo → resaltado

---

#  **FASE 5 — Funcionalidades del menú**

### 10. Inicio
- Limpia la división principal y muestra productos recomendados

### 11. Productos (menú desplegable)
- Lista de productos en la división principal  
- Cada uno abre su ficha individual

### 12. Acerca de
- Texto estático sobre la tienda  
- Cambia según idioma

### 13. Registrarse / Iniciar sesión
- Formulario con 5 campos + regex  
- Guarda sesión en **sessionStorage**

### 14. Carrito
- Guarda productos en **localStorage**  
- Muestra cantidades, precios y total  
- Permite eliminar productos  
- Botón confirmar o cancelar
